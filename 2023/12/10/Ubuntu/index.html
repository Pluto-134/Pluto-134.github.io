Ubuntu

打开Terminal

  1. 右键点击桌面，选择“open in terminal”
  2. Ctrl+Alt+T

在终端中输入

sudo passwd root

如果想获得root权限，只需进行如下的操作

su root

如果要再次禁用 root 帐号，那么可以执行

sudo passwd -l root

列出文件夹下的文件

ls

列出文件夹下的文件，包括隐藏文件

ls -a

列出主文件夹下的包括与文件相关属性的内容信息

ls -l

显示当前目录的绝对路径

pwd

软件安装

sudo apt-get install <package name>
